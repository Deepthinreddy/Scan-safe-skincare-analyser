# Scan & Safe – Skincare Ingredient Analyzer

Scan & Safe is a skincare ingredient analysis tool that helps users understand whether a product is suitable for their skin type.

The app analyzes ingredient lists and detects potentially irritating or pore-clogging ingredients.

## Features

• Ingredient safety analysis  
• Acne trigger detection  
• Personalized warnings based on skin type  
• Product safety scoring  

## Tech Stack

Frontend: HTML, CSS  
Backend: Python (Flask)  
Dataset: JSON ingredient database  

## How It Works

1. User pastes skincare ingredients
2. System analyzes ingredients
3. Product safety score is generated
4. Ingredient warnings are displayed
